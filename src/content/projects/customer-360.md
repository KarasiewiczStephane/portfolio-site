## Overview
Comprehensive customer analytics platform unifying multiple data sources into a single customer view.

## Architecture
- Entity resolution with fuzzy matching
- RFM analysis for behavioral segmentation
- K-Means clustering for customer segments
- BG/NBD + Gamma-Gamma for predictive CLV
- Cohort analysis and retention tracking

## Key Features
- Multi-source data integration
- Automated customer segmentation
- Predictive customer lifetime value
- Interactive cohort analysis dashboard
