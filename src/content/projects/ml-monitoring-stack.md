## Overview
Production ML monitoring solution for detecting model degradation and data drift in real-time.

## Architecture
- Evidently AI for data and model drift detection
- CUSUM and Page-Hinkley for degradation detection
- Prometheus for metrics collection
- Grafana dashboards for visualization
- Failure scenario simulation engine

## Key Features
- Automated drift detection and alerting
- Statistical degradation detection
- Pre-built Grafana dashboards
- Failure scenario simulation for testing
