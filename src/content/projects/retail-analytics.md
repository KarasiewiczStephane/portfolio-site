## Overview
Computer vision system for retail store analytics with privacy-preserving customer tracking.

## Architecture
- YOLOv8 for person detection
- ByteTrack for multi-object tracking
- Zone-based dwell time analysis
- Heatmap generation for traffic patterns
- Privacy-preserving face blurring pipeline

## Key Features
- Real-time foot traffic counting
- Zone-based dwell time analytics
- Heat map visualization
- GDPR-compliant face blurring
